# Climate-aware-agriculture
Climate-Aware Agriculture Prediction App
Overview

Climate-Aware Agriculture Prediction App is a machine learning-based web application developed using Streamlit. The system predicts the most suitable crop and estimates expected crop yield based on climate and soil conditions.

Features
 Crop Recommendation
 Crop Yield Prediction
 Climate Data Analysis
 Interactive Data Visualizations
 User-Friendly Streamlit Dashboard
Technologies Used
Python
Streamlit
Pandas
Scikit-Learn
Matplotlib
Dataset Features

The application uses climate and agricultural parameters such as:

Average Temperature
Rainfall
CO₂ Concentration
Soil Moisture
Fertilizer Usage
Technology Index
Pest Index
How to Run
Clone the repository
Install dependencies:
pip install -r requirements.txt
Run the application:
streamlit run app.py
